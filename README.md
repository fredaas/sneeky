# Sneeky

Play some snake in the terminal while your code is compiling.

<img src="https://github.com/fredaas/sneeky/blob/master/media/screenshots.png" width="100%">

## Usage

Install dependencies:

```
$ ./install.sh
```

Build sneeky:

```
$ make
```

Run sneeky:

```
$ ./bin/sneeky
```

Have fun!
